Installation

Installing the application is straightforward. If you download it from the app store, simply follow the prompts to download and install. If you download it from the official website or other third-party sources, you'll receive a file named v2rayNG_x.x.x.apk, where .apk is the installation package for Android systems, and x.x.x represents the version number. After downloading, click on the file to install it.

Configuration

Step 1

Prepare to add node server information. Click the ➕ button at the top right corner of the app’s main interface to bring up the option to add a configuration file. Refer to the V2rayNG main interface shown above, and add different node server configuration files according to different nodes.

Step 2

You can import server node information either by scanning a QR code or by copying it from the clipboard.

The connection addresses for different protocols of node servers are as follows. Be sure to copy the complete link provided by your service provider.

  • VMESS server, namely v2Ray node address: vmess://

Step 3

After importing the server, click on the "V" icon in the bottom right corner of the app's main interface.

The letter "V" icon initiates the proxy. Upon initial configuration, you will be prompted to create a proxy. Click "OK" to start, as shown in the screenshot below.

Step 4

Once successfully started, the app icon in the bottom right corner of the main interface will change color to indicate successful startup, as shown in the screenshot below.
