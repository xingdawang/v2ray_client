Run and Configure

Step 1

After downloading the software package, decompress it and release it to the local hard disk. Select the v2rayN.exe file and double-click it to run.

Step 1 (Run as Administrator)

Double-click to run as a non-administrator, or right-click to run as an administrator.

Step 2

Click the Server button

Step 3

The easiest way to add a server is from the clipboard or by scanning a QR code on the screen.

After successfully importing the server, find the v2rayN software icon in the system tray in the lower right corner. If you have imported multiple servers, you can choose to use them here.
